# 쇼핑몰 최저가 검색 거비스 "커피두잔"

단국대학교 2021학년도 2학기 실무중심산학협력프로젝트 쇼핑몰별 최저가 검색 서비스 "커피두잔"

## 개요


> 국내 다양한 쇼핑몰의 정보를 모아 최저가를 알려드립니다. 당신의 옆에 "커피두잔".
